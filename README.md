# 파이썬과 컴퓨터 비전 실습 

## 개요

본 자료는 **"파이썬과 컴퓨터 비전"** 단원의 실습 코드와 자료를 담고 있습니다.

## 개발 환경

### 기본 정보
- **IDE**: PyCharm 기반으로 작성
- **Python 버전**: 3.10.10
- **운영체제**: Windows 11

### 주요 라이브러리
```bash
pip install opencv-python
pip install pillow
pip install ultralytics
pip install matplotlib
pip install scikit-image
```

## 프로젝트 구조

